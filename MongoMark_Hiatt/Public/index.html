MongoMark-Hiatt

Use the buttons and radio buttons to demonstrate how to upload a Markdown file, uploading it to, and reading it from a Mongo DB, then translating a JSON object into HTML, to display to the user.

Upload Document

This version of the application only uploads a sample Markdown file called "sample.md". In future implementations, we hope to allow users to select a file they wish to store online. Therefore, the input textbox is read-only.
